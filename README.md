# Simple Todo App

Simple todo app untuk belajar C# dotnet.

## Setup

### Prerequisites

- Visual Studio or Visual Studio Code
- .NET SDK 5.0 or later

### Installation

1. Clone the repository:

   ```bash
   git clone https://github.com/your-username/your-repository.git
   cd your-repository
   ```

2. Restore dependencies:

   ```bash
   dotnet restore
   ```
